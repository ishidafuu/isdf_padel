# B30102-001: サーブが上向き発射され相手コートに届かない

## Status: Done

## 概要

サーブ時にボールが頭上で生成され、斜め上に発射されてすぐ落下してしまう。オーバーハンドサーブとして正しく下向きに打ち下ろされる挙動に修正する。

## 原因

サーブパラメータの設定ミス:
- `serve_angle`: 20度（正の値 = 上向き発射） → **テニスのオーバーハンドサーブは下向きであるべき**
- `serve_speed`: 4.0 m/s（遅すぎる）

## 修正内容

### 設定値変更

| パラメータ | 現在値 | 修正後 |
|-----------|-------|--------|
| `serve_speed` | 4.0 m/s | 10.0 m/s |
| `serve_angle` | 20.0度 | -15.0度 |

### 修正ファイル

1. **設定ファイル**: `project/assets/config/game_config.ron`
2. **デフォルト値**: `project/src/resource/config.rs`
3. **仕様書**: `project/docs/3_ingame/301_match/30102_serve_spec.md`
4. **データ定義**: `project/docs/8_data/80101_game_constants.md`

## 関連仕様

- REQ-30102-060: オーバーハンドサーブ

## 検証方法

1. `cargo run` でゲーム起動
2. サーブ状態でショットボタン（V / ゲームパッドB）を押す
3. ボールが下向きに発射され、相手コートに飛んでいくことを確認
