# B30201-002: コート外移動許可

## 問題

プレイヤーキャラクターがコートの境界線で止まり、コート外に移動できない。
テニスでは選手がコート外に出てボールを追いかけることがあるため、制限を解除する。

## 原因

- `movement.rs` - プレイヤーの移動を自コート内に制限（REQ-30201-004）
- `ai_movement.rs` - AIの移動を自コート内に制限（REQ-30301-004）
- `boundary.rs` - `player_boundary_system`で壁制限（BEH-30503-001, BEH-30503-002）

## 修正内容

1. **movement.rs**: X軸・Z軸の境界クランプを削除、`get_player_x_bounds`関数削除
2. **ai_movement.rs**: X軸・Z軸の境界クランプを削除、`get_ai_x_bounds`関数削除
3. **boundary.rs**: `player_boundary_system`から左右壁制限(BEH-30503-001)と前後壁制限(BEH-30503-002)を削除
   - ネット通過禁止(BEH-30503-003)は維持
   - 天井・地面制限は維持
