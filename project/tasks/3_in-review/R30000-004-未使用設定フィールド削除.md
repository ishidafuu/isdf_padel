|-----------|------|------|

| `PlayerConfig.friction` | v0.2以降使用予定 | 移動システム改善 |
| `PlayerConfig.air_control_factor` | v0.2以降使用予定 | 空中制御 |
| `BallConfig.power_shot_speed` | v0.2使用予定 | ショット属性システム |
| `ShotConfig.jump_shot_angle` | v0.2使用予定 | ショット属性システム |
| `InputConfig.*_buffer_time` | v0.2以降使用予定 | 入力バッファリング |
| `ShotAttributesConfig.push_perfect_window` | v0.2使用予定 | ショット属性 |
| `ShotAttributesConfig.optimal_distance` | v0.2使用予定 | ショット属性 |
