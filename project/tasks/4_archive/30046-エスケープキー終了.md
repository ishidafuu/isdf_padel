# 30046: エスケープキーでウィンドウ終了

## 概要

エスケープキーを押すとゲームウィンドウを閉じる機能を追加。

## 要件

- エスケープキー（Escape）押下でアプリケーション終了
- ゲーム状態に関係なく常に有効

## 実装方針

Bevy の `AppExit` イベントを使用してウィンドウを閉じる。

```rust
fn escape_to_exit(
    keyboard: Res<ButtonInput<KeyCode>>,
    mut exit: EventWriter<AppExit>,
) {
    if keyboard.just_pressed(KeyCode::Escape) {
        exit.send(AppExit::Success);
    }
}
```
