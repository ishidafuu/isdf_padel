# 30052: ネット衝突判定の追加

## 概要

ボールがネットに当たった際の衝突検出・反射・判定を追加する。

## 背景

現在、`NetHitEvent` と判定システム（`let_judgment_system`, `net_fault_judgment_system`）は実装済みだが、ネット衝突を検出してイベントを発行する部分が未実装。

## 実装内容

### 新規ファイル

- `project/src/systems/ball/trajectory/net_collision.rs`
  - ネット衝突検出
  - ボールの跳ね返り処理
  - `NetHitEvent` 発行

### 修正ファイル

- `project/src/systems/ball/trajectory/mod.rs`
  - システム登録

## ネットルール

| 状況 | 結果 |
|------|------|
| サーブ時 → ネットに触れて相手コートに入る | レット（再サーブ） |
| サーブ時 → ネットに触れて相手コートに入らない | フォルト |
| ラリー中 → ネットに触れて相手コートに入る | 続行 |
| ラリー中 → ネットに触れて相手コートに入らない | 失点 |

## 検証方法

`/run-game` で以下を確認：
- 低い軌道でネットに当たると跳ね返る
- 跳ね返ったボールが自コート側に落ちると失点
