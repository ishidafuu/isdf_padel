# R30604-002: パワー計算を加算方式に変更

## 概要
パワー計算を乗算方式から加算方式に変更し、悪い条件でも必ずネットを越える球が打てるようにする。

## 問題
- 現在の乗算方式では、悪い条件が重なると係数が 0.3 × 0.4 × ... ≈ 0.04 となり、パワーが極端に低下
- ニュートラル入力でも自コートに落ちる球になってしまう

## 解決策
**加算方式に変更**:
```rust
// 変更前（乗算方式）
power = base_power * height_factor * timing_factor * distance_factor * approach_factor

// 変更後（加算方式）
power = base_power + height_bonus + timing_bonus + distance_bonus + approach_bonus
```

## 設計方針

| 条件 | ボーナス | 角度補正 | 結果 |
|-----|---------|---------|------|
| 悪い条件 | -2.0〜-3.0 | +20°〜+30° | ただ返すだけの球（山なり） |
| 通常条件 | 0 | 0° | それなりの打球 |
| 良い条件 | +2.0〜+5.0 | -10°〜-15° | 強打（低い弾道） |

## 修正ファイル

3. `/run-game` で実機テスト

   - 良い条件: 低い弾道でオーバーしない
