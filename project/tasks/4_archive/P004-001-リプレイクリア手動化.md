# P004-001: リプレイクリア手動化

## 概要

ビルド時の自動リプレイクリアを廃止し、手動クリアコマンドを提供する。

## 背景

- ビルドのたびにリプレイが消えると意図せずファイルを失う可能性がある
- リプレイのクリアはユーザーが明示的に行うべき

## 実装内容

### 1. build.rs からリプレイクリア処理を削除

**ファイル**: `project/build.rs`

```rust
// リプレイクリア処理を削除
fn main() {}
```

### 2. 手動クリアスクリプトを作成

**ファイル**: `project/scripts/clear-replays.sh`（新規作成）

```bash
#!/bin/bash
SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"
PROJECT_DIR="$(dirname "$SCRIPT_DIR")"
rm -f "$PROJECT_DIR/assets/replays/"*.bin
echo "Replays cleared."
```

## 検証手順

1. `cargo build` でリプレイが消えないことを確認
2. `./scripts/clear-replays.sh` でリプレイが削除されることを確認
