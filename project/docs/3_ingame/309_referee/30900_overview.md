# Referee System Overview

**Version**: 1.0.0
**Status**: Draft
**Last Updated**: 2025-12-25

## 概要

テニス審判システム。得点判定、フォールト判定、サーブ権管理を行います。

## 機能構成

### 1. 得点判定（30901_point_judgment_spec.md）
- アウト判定
- ツーバウンド判定
- ネットイン判定
- コート境界取得

### 2. フォールト判定（30902_fault_spec.md）
- サーブエリア外判定
- ダブルフォルト判定
- Faultカウンタ管理

### 3. サーブ権管理（30903_serve_authority_spec.md）
- サーブ権初期化
- ゲーム終了時の交代
- デュースサイド/アドサイド判定

## MVP v0.1 範囲

以下の機能を実装：
- 基本的な得点判定（アウト、ツーバウンド、ネットイン）
- 基本的なフォールト判定（サーブエリア外、ダブルフォルト）
- 基本的なサーブ権管理（初期化、交代、デュース/アドサイド）

## v0.2以降

以下の機能を追加予定：
- レット判定
- タッチネット判定
- フット・フォルト判定
- 審判コールボイス

## 依存関係

- `304_ball`: ボール状態（位置、バウンド回数）
- `305_court`: コート境界、サーブエリア定義
- `307_scoring`: PointEvent発行、ゲーム終了イベント

## データ参照
