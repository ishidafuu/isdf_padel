# 7_tools Overview

## 概要

開発支援ツール群の仕様書を格納するカテゴリ。

ゲームプレイに直接影響しない、開発・デバッグ・検証のためのツールを定義する。

## サブフォルダ

| フォルダ | 番号 | 内容 |
|---------|------|------|
| 71_simulation | 771 | シミュレーション・自動テストツール |

## 番号体系

- 親番号: 7
- フォルダ番号: 7 × 100 + サブフォルダ番号
- ファイル番号: フォルダ番号 × 100 + 種別番号
