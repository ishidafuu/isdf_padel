# Point Configuration

## 概要

ポイント進行システムの設定データ。

## データ定義

### point_values

テニスのポイント進行値。

| インデックス | 値 | 説明 |
|-------------|-----|------|
| 0 | 0 | 初期値（Love） |
| 1 | 15 | 15 |
| 2 | 30 | 30 |
| 3 | 40 | ゲームポイント |

**参照元**: REQ-30701-001, REQ-30701-002

### point_progression

ポイント進行設定。

| パラメータ | 型 | デフォルト値 | 説明 |
|-----------|-----|-------------|------|
| values | `Vec<u32>` | [0, 15, 30, 40] | ポイント進行値 |
| win_index | `usize` | 4 | この index に到達でゲーム勝利 |

**参照元**: REQ-30701-002, REQ-30701-003

### display_format

ポイント表示形式。

| ポイント値 | 表示 |
|-----------|------|
| 0 | "0" |
| 15 | "15" |
| 30 | "30" |
| 40 | "40" |
| (勝利) | "Game" |

**参照元**: REQ-30701-004

---

## RON設定例

```ron
scoring: ScoringConfig(
    point_values: [0, 15, 30, 40],
    games_to_win_set: 6,
    sets_to_win_match: 1,
),
```
