# Set Configuration

## 概要

セットカウント管理システムの設定データ。

## データ定義

### initial_set_count

セットカウント初期値。

| パラメータ | 型 | デフォルト値 | 説明 |
|-----------|-----|-------------|------|
| player1_sets | `u32` | 0 | Player1のセット数 |
| player2_sets | `u32` | 0 | Player2のセット数 |

**参照元**: REQ-30703-001

### games_to_win_set

セット獲得に必要なゲーム数。

| パラメータ | 型 | デフォルト値 | 説明 |
|-----------|-----|-------------|------|
| games_to_win_set | `u32` | 6 | 6ゲーム先取でセット獲得 |

**参照元**: REQ-30703-002

**MVP v0.1制限**: タイブレークなし（相手のゲーム数に関わらず6ゲーム先取で勝利）

### sets_to_win_match

マッチ勝利に必要なセット数。

| パラメータ | 型 | デフォルト値 | 説明 |
|-----------|-----|-------------|------|
| sets_to_win_match | `u32` | 1 | 1セット先取でマッチ勝利（1セットマッチ） |

**参照元**: REQ-30703-003

**MVP v0.1制限**: 1セットマッチのみ対応

### match_format (v0.2+)

マッチ形式設定（将来拡張）。

| 形式 | 必要セット数 | 説明 |
|------|-------------|------|
| OneSet | 1 | 1セットマッチ（MVP v0.1） |
| BestOfThree | 2 | 3セットマッチ（2セット先取） |
| BestOfFive | 3 | 5セットマッチ（3セット先取） |

**参照元**: REQ-30703-050

---

## RON設定例

```ron
scoring: ScoringConfig(
    point_values: [0, 15, 30, 40],
    games_to_win_set: 6,
    sets_to_win_match: 1,
),
```
