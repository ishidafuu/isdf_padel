## Overview

パデル（Padel）を題材とした **横視点（2.5D）アクションスポーツゲーム**。

壁・天井のある密閉空間で、**ジャンプショット** や **必殺ショット** を駆使して戦う、くにおくんドッジボール風のアクション性を持つパデルゲーム。

## Core Concept

### 1. 壁プレー + アクション性

- **壁・天井反射**: 4面の壁 + 天井、全て反射する（天井もアウトではない）
- **密閉空間**: 完全に密閉されたコート
- **物理法則**: 基本は入射角 = 反射角、ただしバウンドは強め調整（ラリー継続のため）

### 2. アクション要素

- **ジャンプショット**: ジャンプ中にショット可能
- **必殺ショット**: 適切な高さで打つことで発生（詳細は未定、実装しながら固める）
- **キャラクター当たり判定**: ボールがキャラクターに直接当たる
- **ふっとばし**: ボールが当たるとキャラクターが吹っ飛ぶ

### 3. 2.5D表現

- **X軸**: 左右移動（画面の横方向）
- **Y軸**: 奥行き移動（無段階、くにおくん方式）
- **Z軸**: 高さ（ジャンプ）
- **擬似3D**: 完全な2Dではなく、2.5D表現

## Controls

| 操作 | ボタン | 備考 |
|------|--------|------|
| 左右移動 | 方向キー（左右） | X軸移動 |
| 奥行き移動 | 方向キー（上下） | Y軸移動（無段階） |
| ジャンプ | **Aボタン** | Z軸移動 |
| ショット | **Bボタン** | ボールを打つ |
| ダッシュ | **未定** | 後ほど検討 |

**重要**: 当初のテニス想定（A=ショット）と逆になりました。
A=ジャンプ、B=ショット。

## Court Structure

### サイズ

- **約10m x 20m**（テニスより小さい）
- **密閉空間**: 4面の壁 + 天井

### 壁・天井

- **左右の壁**: X軸方向の境界
- **前後の壁**: Y軸方向の境界（画面手前側も壁）
- **天井**: 壁の上部、**反射する**（アウトではない）

### 反射ルール

- **基本**: 入射角 = 反射角
- **調整**: バウンドは強めに調整（ラリー継続のため）
