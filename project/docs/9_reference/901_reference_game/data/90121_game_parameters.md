## データの活用方法

### 確度★★★★★（確認済）
- そのまま採用可能

### 確度★★★☆☆〜★★★★☆（中〜高確度）
- 参考として採用、要調整

### 確度★☆☆☆☆〜★★☆☆☆（低確度）
- 参考程度、実測・検証必須

## 不明データの優先度

実装時に実測・検証が必要な優先度の高いパラメータ：

1. **プレイヤー移動速度** - ゲーム感に直結
2. **ボール速度** - ゲーム難易度に直結
3. **タイミング判定猶予** - 操作感に直結
4. **AIパラメータ** - ゲームバランスに直結
5. **コートサイズ** - レイアウトに影響

## 参照資料リンク（spec-agent 用）

本データを新規仕様に反映する際の参照：

```
**参考**: ゲームパラメータ（本ドキュメント）
- 確度: ★★☆☆☆（推測が多い）
- 参照推奨項目: テニスルール部分のみ（★★★★★）、他は参考程度
```

**重要**: このファイルのデータは推測が多いため、実装時には必ず実測・検証を行うこと。
