# ショット操作

## 概要

ボールを打ち返す操作。Aボタン・Bボタンを使い分けることで、異なる種類のショットを打つことができる。

## 観察された動作

### ショットの種類

#### サーブ
- **トリガー**: サーブ時にボタン入力
- **結果**: ボールを相手コートに打ち込む
- **確度**: ★★★★☆（複数の攻略サイトで確認）
- **備考**: タイミングが重要

#### ストローク（Aボタン）
- **トリガー**: ボールが近くにある時にAボタン
- **結果**: 通常の返球
- **確度**: ★★★★☆（攻略サイト記載）
- **特徴**: 基本的な返球ショット

#### ロブ・スマッシュ（Bボタン）
- **トリガー**: ボールが近くにある時にBボタン
- **結果**: 高い軌道のロブ、または強打のスマッシュ
- **確度**: ★★★☆☆（Web情報から推測）
- **備考**: ボールの高さやタイミングで使い分け

### タイミング判定
- **判定**: ボールとの距離・タイミングでショット成否が決まる
- **確度**: ★★★★☆（ゲーム動画から観察）
- **ミス条件**: タイミングが合わないとミスショット

### 打ち分け
- **方向指定**: 十字キーでショットの方向を指定可能
- **確度**: ★★★☆☆（攻略情報から推測）
- **備考**: 詳細な角度制御の有無は不明

## 操作仕様

| 操作 | ショット種類 | 特徴 |
|------|-----------|------|
| Aボタン | ストローク | 通常の返球 |
| Bボタン | ロブ/スマッシュ | 高軌道または強打 |
| 十字キー+A/B | 方向指定ショット | 打ち分け |

## ショットの軌道

### ストローク
- **軌道**: 中程度の高さ
- **速度**: 中程度
- **確度**: ★★☆☆☆（動画観察）

### ロブ
- **軌道**: 高い弧を描く
- **速度**: 遅い
- **確度**: ★★☆☆☆（動画観察）

### スマッシュ
- **軌道**: 急角度
- **速度**: 速い
- **確度**: ★★☆☆☆（動画観察）

## 数値データ（判明している範囲）

| パラメータ | 値 | 確度 |
|-----------|----|----|
| ショット種類数 | 2～3種類 | ★★★☆☆ |
| タイミング許容フレーム | 不明 | ★☆☆☆☆ |
| 方向指定精度 | 不明 | ★☆☆☆☆ |

## ゲームプレイへの影響

- **戦略性**: ショットの使い分けで戦略的なプレイが可能
- **難易度**: タイミング判定がゲームの難易度を決定
- **操作感**: シンプルながら奥深い操作性

## 不明点

- 各ショットの正確な速度・軌道パラメータ
- タイミング判定の具体的なフレーム数
- スピンの有無
- ボレーとグランドストロークの区別
- ドロップショットなどの特殊ショット
