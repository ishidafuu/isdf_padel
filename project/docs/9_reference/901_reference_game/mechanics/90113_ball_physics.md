# ボール物理

## 概要

テニスボールの挙動を制御する物理システム。ボールの軌道、速度、バウンドなどを管理する。

## 観察された動作

### ボールの軌道
- **パターン**: 放物線を描いて飛ぶ
- **確度**: ★★★★☆（ゲーム画面から明確に確認）
- **備考**: シンプルな2D放物線運動

### バウンド
- **挙動**: コートに着地すると跳ね返る
- **バウンド回数**: 1回バウンド後にプレイヤーが打つ
- **確度**: ★★★★☆（テニスルールから推測、動画で確認）

### ネット判定
- **判定**: ボールがネットに当たると落下
- **結果**: 相手側にボールが届かずポイント喪失
- **確度**: ★★★★☆（テニスルールから確認）

### アウト判定
- **判定**: ボールがコート外に落下
- **結果**: ポイント喪失
- **確度**: ★★★★☆（テニスルールから確認）

### ボールの速度
- **変化**: ショットの種類によって速度が変わる
- **確度**: ★★★☆☆（動画観察から推測）
- **詳細**: ストローク < ロブ（遅い）、スマッシュ（速い）

## ボールの表示

### 視覚表現
- **サイズ**: 小さな円形
- **カラー**: 白または黄色（推測）
- **確度**: ★★★☆☆（レトロゲーム画面から推測）

### 影表示
- **有無**: ボールの下に影が表示される（推測）
- **確度**: ★★☆☆☆（同時期のゲームから推測）
- **目的**: ボールの高さを視覚的に表現

## 物理パラメータ（推測）

| パラメータ | 推測値 | 確度 |
|-----------|--------|------|
| 重力加速度 | あり（放物線） | ★★★★☆ |
| 空気抵抗 | なし（簡易物理） | ★★☆☆☆ |
| バウンド係数 | 中程度 | ★★☆☆☆ |
| 初速度範囲 | ショット種類依存 | ★★☆☆☆ |

## ゲームプレイへの影響

- **予測**: ボールの軌道を読んで移動する必要がある
- **タイミング**: ボールのバウンドタイミングでショット
- **戦略**: ボールの速度と軌道で攻防を展開

## コート判定

### インコート
- **範囲**: シングルスコートの境界線内
- **確度**: ★★★★☆（テニスルールから確認）

### アウト
- **範囲**: コート境界線外
- **確度**: ★★★★☆（テニスルールから確認）

### ネット
- **高さ**: ネットの高さ以下で接触するとアウト
- **確度**: ★★★★☆（テニスルールから確認）

## 不明点

- ボールの正確な速度パラメータ（ピクセル/フレーム）
- バウンド係数の具体的な数値
- スピンの実装有無
- ボールの回転表現
- 風や気象条件の影響
- 正確な重力加速度の値
