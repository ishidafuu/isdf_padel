# AI挙動

## 概要

1人プレイ時のコンピュータ対戦相手（AI）の動作パターン。プレイヤーの操作に対して適切に反応し、テニスの試合を成立させる。

## 観察された動作

### 基本動作パターン

#### ボール追跡
- **挙動**: ボールの落下地点に移動して返球を試みる
- **確度**: ★★★★☆（ゲーム動画から観察）
- **備考**: プレイヤーと同様の移動メカニクスを使用（推測）

#### 返球判断
- **挙動**: ボールに到達するとショットを打つ
- **確度**: ★★★☆☆（動画観察から推測）
- **備考**: タイミング判定はプレイヤーと同様（推測）

### ショット選択

#### ストローク
- **使用頻度**: 最も頻繁に使用
- **確度**: ★★★☆☆（動画観察）

#### ロブ・スマッシュ
- **使用頻度**: 状況に応じて使用
- **確度**: ★★☆☆☆（動画観察、詳細不明）

### 難易度による変化

#### 動作速度
- **変化**: 難易度が上がると移動・反応速度が向上（推測）
- **確度**: ★★☆☆☆（一般的なゲーム仕様から推測）

#### ミス率
- **変化**: 難易度が上がるとミスが減る（推測）
- **確度**: ★★☆☆☆（一般的なゲーム仕様から推測）

#### 戦術性
- **変化**: 高難易度ではより戦略的な配球（推測）
- **確度**: ★☆☆☆☆（詳細不明）

## AI思考パターン（推測）

### 移動判断
1. ボールの軌道を予測
2. 落下地点を計算
3. 最短ルートで移動開始

### 返球判断
1. ボールとの距離を計算
2. タイミング判定
3. ショット種類を選択
4. 方向を決定

**確度**: ★☆☆☆☆（完全な推測）

## 観察可能な特徴

- **基本的な返球**: 確実にボールを追いかけて返球を試みる
- **反応性**: プレイヤーのショットに対して反応する
- **難易度差**: 難易度によって強さが変わる（推測）

## ゲームプレイへの影響

- **練習相手**: 1人プレイ時の対戦相手として機能
- **難易度調整**: プレイヤーのスキルに応じた難易度設定
- **挑戦**: 上級AIの攻略が目標
