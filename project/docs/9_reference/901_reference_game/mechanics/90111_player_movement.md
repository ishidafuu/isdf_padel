# プレイヤー移動

## 概要

コート内でプレイヤーキャラクターを移動させる基本操作。十字キーでコート上を自由に移動できる。

## 観察された動作

### 基本移動
- **トリガー**: 十字キー入力
- **結果**: キャラクターが入力方向に移動
- **確度**: ★★★★☆（複数の攻略サイトで確認）
- **備考**: 8方向移動が可能

### 移動速度
- **値**: 一定速度で移動
- **確度**: ★★☆☆☆（動画観察からの推測）
- **備考**: ボールとの距離や状況による速度変化は確認できず

### 移動範囲
- **範囲**: コート内全域
- **制限**: コートの境界線で停止
- **確度**: ★★★★☆（ゲーム画面から確認）

### 移動中の挙動
- **アニメーション**: 走るモーション
- **確度**: ★★★★☆（画像・動画から確認）

## 操作仕様

| 入力 | 動作 | 備考 |
|------|------|------|
| 十字キー上 | 画面奥方向に移動 | ネット側 |
| 十字キー下 | 画面手前方向に移動 | ベースライン側 |
| 十字キー左 | 画面左方向に移動 | - |
| 十字キー右 | 画面右方向に移動 | - |
| 斜め入力 | 斜め方向に移動 | 8方向移動 |

## 数値データ（判明している範囲）

| パラメータ | 値 | 確度 |
|-----------|----|----|
| 移動方向 | 8方向 | ★★★★☆ |
| 移動速度 | 不明 | ★☆☆☆☆ |
| 加速度 | なし（即座に最高速） | ★★☆☆☆ |

## ゲームプレイへの影響

- **ボール追跡**: ボールの落下地点に素早く移動することが重要
- **ポジショニング**: コート内での位置取りが戦略的要素
- **返球準備**: 移動完了後にショット操作に移行

## 不明点

- 正確な移動速度（ピクセル/フレーム単位）
- 移動中の慣性の有無
- ダッシュやスライディングなどの特殊移動の有無
