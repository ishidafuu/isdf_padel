# 試合画面（メインゲーム画面）

## 概要

実際のテニスの試合が行われるメインゲーム画面。コート、プレイヤー、ボール、スコアなどが表示される。

## 画面レイアウト

### 視点
- **視点**: トップビュー（俯瞰視点）
- **確度**: ★★★★☆（ゲーム画面から確認）
- **特徴**: コート全体を上から見下ろす視点

### コート表示
- **要素**: テニスコート（シングルス）
- **境界線**: コートの境界線が描画
- **ネット**: 中央にネット
- **確度**: ★★★★☆（ゲーム画面から確認）

### プレイヤー表示

#### 自プレイヤー
- **位置**: 画面下側（ベースライン側）
- **表示**: デフォルメされた人型キャラクター
- **確度**: ★★★★☆（ゲーム画面から確認）

#### 相手プレイヤー
- **位置**: 画面上側（ネット向こう側）
- **表示**: 同様のキャラクター表示
- **確度**: ★★★★☆（ゲーム画面から確認）

### ボール表示
- **表示**: 小さな円形オブジェクト
- **影**: ボールの下に影（推測）
- **確度**: ★★★☆☆（レトロゲーム画面から推測）

## UI要素

### スコア表示
- **位置**: 画面上部または下部
- **内容**:
  - 現在のポイント（0, 15, 30, 40）
  - ゲームカウント
  - セットカウント（推測）
- **確度**: ★★★☆☆（一般的なテニスゲームから推測）

### サーブ表示
- **表示**: サーブ権の表示（推測）
- **確度**: ★★☆☆☆（詳細不明）

## 画面遷移

### 遷移元
- モード選択画面

### 遷移先
- リザルト画面（試合終了時）
- タイトル画面（中断時、推測）

## 色使い（推測）

| 要素 | 色 | 確度 |
|------|----|----|
| コート | 緑系 | ★★☆☆☆ |
| 境界線 | 白 | ★★★☆☆ |
| ネット | 白 | ★★★☆☆ |
| プレイヤー | カラフル | ★☆☆☆☆ |
| ボール | 白/黄 | ★★☆☆☆ |

**注意**: ファミコンの色数制限内での表現

## 操作系

### 移動
- **入力**: 十字キー
- **結果**: プレイヤーキャラクターが移動

### ショット
- **入力**: A/Bボタン
- **結果**: ボールを打つ

詳細は以下を参照：
- [プレイヤー移動](../mechanics/90111_player_movement.md)
- [ショット操作](../mechanics/90112_shot_system.md)

## ゲームフロー

1. **サーブ**: サーバーがサーブを打つ
2. **ラリー**: ボールを打ち合う
3. **ポイント決着**: どちらかがミス
4. **スコア更新**: ポイントが加算
5. **次のポイント**: サーブから再開
6. **試合終了**: 規定のセット数を獲得

## 画面上の情報密度

- **シンプル**: 必要最小限の情報
- **視認性**: プレイに集中できるレイアウト
- **確度**: ★★★☆☆（レトロゲームの一般的な設計思想）

## 不明点

- 正確なUIレイアウト（スコア表示位置等）
- サーブ表示の有無と位置
- 風向き表示の有無
- リプレイ機能の有無
- ポーズメニューの内容
- 効果音・BGMの有無
