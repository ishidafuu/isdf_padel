# リザルト画面

## 概要

試合終了後に表示される結果画面。勝敗とスコアを表示。

## 表示要素

### 勝敗表示
- **内容**: WIN / LOSE
- **確度**: ★★★☆☆（一般的なゲームから推測）

### スコア表示
- **内容**: 最終スコア（セットカウント等）
- **確度**: ★★☆☆☆（詳細不明）

## 画面遷移

- **前の画面**: 試合画面
- **次の画面**: タイトル画面、またはモード選択画面

## 不明点

- 詳細なスコア表示内容
- 統計情報の有無
- リプレイオプションの有無
