# モード選択画面

## 概要

1人プレイか2人プレイかを選択する画面。

## 表示要素

### モード選択
- **1 PLAYER**: プレイヤー vs コンピュータ
- **2 PLAYERS**: プレイヤー vs プレイヤー
- **確度**: ★★★★☆（標準的なゲームモードから推測）

## 難易度選択

- **有無**: 1人プレイ時に難易度選択が可能（推測）
- **確度**: ★★☆☆☆（詳細不明）

## 画面遷移

- **前の画面**: タイトル画面
- **次の画面**: 試合画面

## 不明点

- 正確なモード数
- 難易度選択の段階数
- キャラクター選択の有無
