# タイトル画面

## 概要

ゲーム起動時に表示される最初の画面。ゲームタイトルとスタート選択肢を表示。

## 表示要素

### タイトルロゴ
- **内容**: 「ナムコファミリーテニス」
- **確度**: ★★★★☆

### 開始メニュー
- **選択肢**: START等の開始オプション
- **確度**: ★★★☆☆（一般的なファミコンゲームから推測）

## 画面遷移

- **次の画面**: モード選択画面
- **入力**: STARTボタン

## 不明点

- デモ画面の有無
- BGMの有無
- オプション設定画面の有無
