# Task F024: 禁止事項番号重複の修正

## 説明

複数のエージェントファイルで禁止事項の番号が重複している問題を修正する。

## 背景

### 現状

約8ファイルで禁止事項の番号「2.」が2回出現:
```markdown
2. **仕様書の直接修正**
...
2. **実装コードの批評**  ← 3. であるべき
```

### 改修理由

- 番号の重複は参照時の混乱を招く
- ドキュメント品質の向上
