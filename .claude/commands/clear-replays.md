# /clear-replays コマンド

リプレイフォルダ内のファイルを削除します。

## 使用者

**👤 人間向けコマンド** - 不要なリプレイファイルの削除に使用

## 使用方法

```
/clear-replays
```

## 指示

1. `project/scripts/clear-replays.sh` を実行
2. 削除結果を報告

### 実行コマンド

```bash
/Users/ishidafuu/Documents/repository/isdf_padel/project/scripts/clear-replays.sh
```

## 注意事項

- 削除したリプレイファイルは復元できません
- assets/replays/ 内の .bin ファイルのみが削除対象です
